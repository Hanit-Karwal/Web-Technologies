HTML TABLES WITH LINKS AND LISTS

Unordered HTML List:
An unordered list starts with the ul tag. Each list item starts with the li tag.The list items will be marked with bullets (small black circles) by default as shown in flipkart row.
Ordered HTML List:
An ordered list starts with the ol tag. Each list item starts with the li tag. The list items will be marked with numbers by default as shown in amazon row.
HTML Description Lists:
HTML also supports description lists.A description list is a list of terms, with a description of each term. The dl tag defines the description list, the dt tag defines the term (name), and the dd tag describes each term.
